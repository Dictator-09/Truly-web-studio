# Truly Web Studio

A boutique creative studio specializing in high-end digital experiences, with a focus on 'Atmospheric Monolith' aesthetics and smooth, elite performance.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- GSAP for premium animations
- Framer Motion

## Development
To start the local development server:

```bash
npm run dev
```

## Production
To create an optimized production build:

```bash
npm run build
npm start
```
