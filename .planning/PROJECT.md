# Truly Web - Creative Studio Site

A premier, high-end creative studio website featuring fluid animations, minimalist aesthetics, and a premium user experience.

## Goal
Build a world-class creative studio landing page that "wows" users with fluid particle physics and a cinematic UI layout.

## Core Values
- **Fluidity**: Every interaction should feel organic and smooth.
- **Impact**: High-contrast, bold typography and generous spacing.
- **Glassmorphism**: Subtle, premium transparency effects.
